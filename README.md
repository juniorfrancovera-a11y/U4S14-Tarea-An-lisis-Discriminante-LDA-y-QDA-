# U4S14-Tarea-An-lisis-Discriminante-LDA-y-QDA-
# Comparación entre LDA y QDA utilizando el Wine Dataset

## Descripción

Este Tarea tiene como objetivo comparar el desempeño del Análisis Discriminante Lineal (LDA) y el Análisis Discriminante Cuadrático (QDA) utilizando el Wine Dataset de la biblioteca Scikit-learn.

Se desarrolló un análisis exploratorio de los datos, la implementación de ambos modelos de clasificación y una comparación de sus resultados mediante diferentes métricas de evaluación y la visualización de sus fronteras de decisión.

## Dataset utilizado

Se utilizó el Wine Dataset incluido en la biblioteca Scikit-learn.

Características:

- 178 observaciones.
- 13 variables predictoras.
- 1 variable objetivo.
- 3 clases.

## Tecnologías utilizadas

- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Cómo ejecutar el proyecto

1. Abrir el notebook en Google Colab.
2. Ejecutar todas las celdas.
3. El Wine Dataset se carga automáticamente desde Scikit-learn.

## Resultados principales

- LDA obtuvo un Accuracy de 100 %.
- QDA obtuvo un Accuracy de 97.22 %.
- LDA presentó el mejor desempeño para este conjunto de datos.
